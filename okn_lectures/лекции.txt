Шаблоны - инструмент для компилятора, 
позволяющий создавать перегруженные функции 
в автоматическом режиме в зависимости от способа вызова функции.

сигнатура функции


printf types:
// %d - int
// %f - float
// %lf - double
// %c - char
// %s - string
// %ld - long long
// %p - pointer
// %x - вывод int в 16-ричной СС
// %g - вывод вещественных целых в компактной записи (без запятой, если оно целое)


Трансляция - процесс перевода текстого файла в машинный код
1. Препроцессинг (команды, выполняемые перед компиляцией; начинающиеся с '#')
2. Компиляция (поиск ошибок, процесс выполняется раздельно по .cpp файлам) (ошибки C*)
3. Сборка (linking: набор объектных файлов -> исполняемый файл) (ошибки LNK*)

#define NAME VALUE - заменяет все вхождения NAME на VALUE
#include <...> - объединяет несколько файлов в один
